Role And Privilege (Permission)
===================================

Authority 를  Role 과 Privilege 로 구분하여, 역할 별 권한을 동적으로 구성 가능하게 한다.
- Role: 사용자에 부여하는 역할. 서비스에서 동적으로 생성, 구성 가능
- Privilege(Permission): 시스템에 정의된 권한.
